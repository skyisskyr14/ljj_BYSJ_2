## Project goal
Build a production-like graduation project:
基于SpringBoot+Vue的云同学录系统

## Non-negotiable rules
- Do not output pseudo code.
- Do not leave TODOs for core features.
- Do not generate ugly admin-template-only UI.
- Frontend must look modern, clean, friendly, and product-like.
- All main buttons and forms must be wired to real logic.
- Keep naming consistent and code maintainable.
- Prefer stable, mainstream solutions suitable for student defense/demo.

## Tech defaults
- Frontend: Vue 3 + Vite + TypeScript + Pinia + Vue Router + Element Plus
- Backend: Spring Boot 3 + Spring Security + JWT + MyBatis-Plus
- Database: MySQL 8
- Cache: Redis
- Build: Maven

## What “done” means
- Backend APIs implemented
- Frontend pages implemented
- Login/auth/rbac works
- CRUD works
- SQL schema provided
- Seed data provided
- README provided
- Default admin/user accounts provided
- Project can run locally

## UI expectations
- Modern visual hierarchy
- Consistent spacing, radius, shadows, typography
- Good empty/loading/error states
- Responsive enough for common screens
- Login, home, profile, directory, detail, message wall, admin pages all polished

## Coding expectations
- Clear module boundaries
- Reusable components
- Clean API layer
- Unified response wrapper
- Global exception handling
- Form validation
- Operation logs
- Sensible comments where needed
